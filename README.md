Mashery-PHP-API Bundle
======================

Bundle built for PHP API(v2) library for [Mashery](http://www.mashery.com)

***Resources:***

- [API Specifications](http://support.mashery.com/docs/read/mashery_api/20)

Requirements
============
- PHP 5.4 and higher
- Curl extension
